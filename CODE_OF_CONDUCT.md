# Code of Conduct

## Our Standards

Contributors are expected to:

1. Be respectful and constructive in communication.
2. Focus feedback on ideas, code, and outcomes.
3. Avoid harassment, discrimination, and personal attacks.
4. Assume good intent while still enforcing quality standards.

Unacceptable behavior includes:

1. Harassment or intimidation.
2. Abusive or insulting language.
3. Publishing private information without permission.
4. Deliberate disruption of collaborative work.

## Scope

This Code of Conduct applies in:

1. Repository issues, pull requests, and discussions.
2. Project-related chats and review channels.
3. Any public spaces representing this project.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this policy.

Responses to violations may include:

1. Warning and request for corrective action.
2. Temporary or permanent removal from collaboration spaces.
3. Content moderation or removal.

## Reporting

If you experience or witness unacceptable behavior, report it to maintainers through repository communication channels. Security-sensitive incidents should follow [SECURITY.md](SECURITY.md).
