# Employment Issue Frameworks

Version: `1`

Status: `approved_for_implementation`

This document defines the neutral employment-matter issue tracks introduced for Milestone 7.3.

These tracks are:

- issue-spotting frameworks
- evidence-organization frameworks
- intake and report structures for HR or counsel-facing review

They are not:

- legal holdings
- statutory-satisfaction claims
- liability findings

## Current issue tracks

- `disability_disadvantage`
- `retaliation_after_protected_event`
- `eingruppierung_dispute`
- `prevention_duty_gap`
- `participation_duty_gap`

## Shared rules

- every issue track must keep a neutral framing question
- every issue track must list required proof elements
- every issue track must list normal alternative explanations
- every issue track must list missing-document needs
- every issue track must list minimum source-quality expectations
- the report must distinguish:
  - `supported_by_current_record`
  - `alleged_but_not_yet_evidenced`

## Status semantics

`supported_by_current_record` means the supplied record supports continued review of that issue track without implying that a legal threshold is met.

`alleged_but_not_yet_evidenced` means the issue track is in scope, but the current record still lacks enough issue-specific support or documentation for stronger treatment.

## Track summaries

### `disability_disadvantage`

Use when the record may involve illness- or disability-linked disadvantage, exclusion, or unequal treatment.

### `retaliation_after_protected_event`

Use when a complaint, objection, disclosure, or participation act may have been followed by worse treatment.

### `eingruppierung_dispute`

Use when the record points to disputed task level, role allocation, tariff grouping, or classification handling.

### `prevention_duty_gap`

Use when the record may involve missing prevention-oriented follow-up, BEM-like process steps, or unexplained gaps around workability support.

### `participation_duty_gap`

Use when the record may involve missing or bypassed participation by SBV, Personalrat, or comparable consultation paths.
